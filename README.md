# E-Commerce Platform

A modern and scalable eCommerce platform designed to help businesses sell products online with a seamless shopping experience, secure payment processing, inventory management, and powerful administrative controls.

---

## Overview

This platform provides everything required to launch and manage a successful online store. From product catalog management and customer accounts to order processing and analytics, the system is built to support businesses of all sizes.

Whether you're running a single-vendor store, dropshipping business, or expanding into a multi-vendor marketplace, this platform offers the flexibility and scalability needed for growth.

## Installation

### Clone Repository

```bash
git clone https://github.com/yourusername/ecommerce-platform.git
```

### Install Dependencies

```bash
npm install
```

### Setup Environment Variables

```bash
cp .env.example .env
```

### Start Development Server

```bash
npm run dev
```

### Build Production Version

```bash
npm run build
```

### Run Production

```bash
npm run start
```
